# Portfolio

For this project, I focused on improving both the design and usability of my portfolio website. I spent time adjusting the colors, fonts, and spacing to make everything look more clean and professional, instead of just basic or thrown together. One of the main things I worked on was responsiveness, making sure the site looked good on both my laptop and my phone, since I realized how different layouts can break on smaller screens. I also stayed consistent with pushing my work to GitHub so I wouldn’t lose progress and could track my changes. Deploying the site was honestly one of the more challenging parts, but it helped me understand how everything connects from local files to a live website. Overall, this project helped me see the importance of not just making something work, but making it look good and easy for users to navigate.
